## ğŸ§  How It Works

1. User enters skills or uploads a resume  
2. Text is cleaned and normalized  
3. TF-IDF vectorization converts text to numerical vectors  
4. Cosine similarity computes similarity between skills and job requirements  
5. Jobs are ranked based on similarity score  
6. Results are displayed visually
